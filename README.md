Employee Salary Automation Bot
About Project
This is a UiPath automation project created for processing employee salary details using Excel files.
Features
- Read employee data from Excel
- Process salary details
- Filter high salary employees
- Generate CSV output file

Tools Used
- UiPath Studio
- Excel
- CSV File

Workflow
1. Read Excel data
2. Process employee salary
3. Filter required data
4. Create output report

Output
The project generates a CSV file with processed employee salary details.
